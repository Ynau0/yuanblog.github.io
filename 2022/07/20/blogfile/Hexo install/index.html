什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装准备

安装Hexo前需要以下两个程序：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

如果您不知道电脑上是否已经安装了这两个程序，您可以进行如下操作进行检查：

  • 以Windows为例，打开 cmd 命令提示符，快捷打开方式为键盘的 Win 键+R，打开运行窗口，输入 cmd 回车进入 cmd 界面，在
  • 在上述的 > 后面输入 node -v 或者 node –version

该指令是检查 node 的版本号，如果您电脑已经安装了 node 则 cmd 上会出现 Node 的版本号，否则您的电脑尚未安装 Node，需要进行 Node.js安装教程。

同样的需要查看 Git 是否已经在电脑上完成安装，可以在cmd中输入 git -v 进行查看
